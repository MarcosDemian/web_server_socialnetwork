# Web Server en Node.JS

Practica de servidor web echo con Node.JS y Express donde es servida una aplicación tipo red social

## ¿Cómo usarlo?

 1. Debes realizar el clonado del repositorio `git clone https://github.com/MarcosDemian/web_server_socialnetwork`.
 2. Abrir en tu editor de código y en terminal ejecutar el comando `npm install`.
 3. Personaliza tu variable de entorno `.env` con la plantilla dada `.env.template`. (Recomendado PORT:3000 y carpeta: `public`)
 4. Ya puedes ejecuta el comando en la terminal `npm run dev`.
